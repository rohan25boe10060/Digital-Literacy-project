Module Summaries
Task 1 – Digital Literacy Infographic (Module 1)
Created a one-page infographic using Canva covering the definition of digital literacy, digital tools for students, and safe internet practices. The visual was designed to be shared with batchmates as an awareness resource.
Task 2 – Student Digital Portfolio (Module 2)
Set up professional profiles on GitHub, LinkedIn, and Kaggle. Created a GitHub Profile README, filled in LinkedIn Education, and set up a Kaggle beginner profile. These platforms will be my digital footprint throughout my engineering journey.
Task 3 – Coding & Collaboration Platforms (Module 3)
Completed the HackerRank "Solve Me First" beginner challenge and created a Google Form for a Digital Literacy Awareness Quiz with 5 questions (including MCQ and short answer formats).
🔗 Digital Literacy Quiz (Google Form): https://forms.gle/digitalquiz2024
Task 4 – Email Etiquette & Social Media Guide (Module 4)
Drafted two professional emails (assignment extension request and internship inquiry) and created a comprehensive Social Media Do's and Don'ts checklist for college students.
Task 5 – Cybercrime Awareness (Module 5)
Researched UPI/Online Payment Fraud and wrote a detailed case study with a step-by-step breakdown of how it occurs. Produced an 8-point prevention checklist targeted at college students in India.
